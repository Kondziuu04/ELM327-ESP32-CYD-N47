A compact OBD-II data display system that shows real-time vehicle diagnostics on a TFT screen. This project connects to your car's OBD-II port via an ELM327 Bluetooth adapter and displays key metrics like:

-Engine coolant temperature

-Engine oil temperature

-Manifold pressure (for turbo/supercharged engines)

-Battery voltage

Features:

Dual Protocol Support:

-Standard OBD-II PIDs

-BMW-specific PIDs (automatically detected)(tested on BMW E87 118d)
